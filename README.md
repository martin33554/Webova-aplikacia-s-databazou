# Školský projekt: Webová aplikácia s databázou

Tento projekt bol zameraný na implementáciu backend logiky, autentifikácie používateľov a prepojenie webovej aplikácie s relačnou databázou (MySQL). 

*Poznámka: Frontend (HTML/CSS šablóna) bol poskytnutý v rámci zadania ako základ. Mojou úlohou bolo kompletne navrhnúť a implementovať backendovú časť.*

## Moja implementácia
- **Registrácia a prihlasovanie:** Bezpečné overovanie používateľov.
- **Prepojenie s databázou:** Návrh databázovej štruktúry a implementácia SQL dopytov.
- **Zabezpečenie:** Základná ochrana proti SQL injection a XSS útokom.

## Použité technológie
- PHP
- MySQL / SQL
- HTML, CSS (poskytnutá šablóna)
